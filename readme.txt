Chek Creative reCAPTCHA validation for client forms

== Description ==

This plugin is intended for Chek Creative-hosted WordPress sites. It adds reCAPTCHA validation for client forms submitting to Zapier.


== Installation ==

1. Install plugin via Chek Creative ([hello@chekcreative.com](mailto:hello@chekcreative.com)) support.
